轻量级数据库SharedPreferences的用法

SharedPreferences 这个数据库，感觉还是比较好用的，他一般用来存储一些配置信息。下面我来写下SharedPreferences的创建及应用
创建一个SharedPreferences数据库（数据库的名字为file）：
SharedPreferences mySharedPreferences= getSharedPreferences(“file”,
Activity.MODE_PRIVATE);
//Activity.MODE_PRIVATE 这是个自带常量
//Activity.MODE_PRIVATE 属性该文件只能被创建他的应用访问（控制访问权限）
//Activity.MODE_APPEND 的功能是判断是否有该文件，如果有在后面添加，而不是擦除
//实例化SharedPreferences.Editor对象
SharedPreferences.Editor editor = mySharedPreferences.edit();

//数据的存入 用putString的方法保存数据
editor.putString(“file_name”, “ying”);

//提交当前数据
editor.commit();

//获取存入的file_name 这个key下的数据
String myname = mySharedPreferences.getString(“file_name”, “”);
